In Clasa Catalog am o lista de documente, constructorul normal si un constructor gol, o metoda addDocument ce imi adauga documente in lista
si get-ul pentru documents, findById ce returneaza un document dupa id si metoda toString 
In Clasa Document am atributele necesare, get si set-urile necesare, constructorul, o mapa si metodele get si set ale ei
In clasa CatalogUtil am metoda save care adauga un catalog intr-un fisier si metoda load care preia din fisier un catalog
In clasa InvalidCatalogException am o exceptie 
In Main adaug 2 documente in catalog, salvez catalogul in fisier si dupa ii dau load.
